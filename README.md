# StudyFlow

StudyFlow is a simple study management web application created using ASP.NET Web Forms style pages.

## Features

- Public home page
- Public learning resources page
- About page
- Create account page
- Login page using localStorage
- Dashboard
- Saved resources page
- Study plans page
- Progress tracker
- Daily streak page
- User profile page

## Technologies Used

- ASP.NET Web Forms style pages
- HTML
- CSS
- JavaScript
- LocalStorage

## Note

This project was developed as a front-end prototype.  
User account data is stored in browser localStorage for demonstration purposes.
